EMAI-COPD
=========

Ethics-aware Multimodal AI for COPD

EMAI-COPD is a multi-tier framework for organizing data preparation, unimodal baselines,
evaluation, and future multimodal modeling for COPD research.

Highlights
----------

- Tier 1: data preparation, dataset cards, general tabular models, evaluation
- Tier 2: multimodal modeling and integration
- Tier 3: advanced system-level and translational components

Current Focus
-------------

The current public documentation focuses on Tier 1, including modality-specific datasets
and benchmark model families for structured prediction tasks.

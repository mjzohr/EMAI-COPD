Model Catalog
=============

Summary
-------

Tier 1 model selection includes baseline, seminal, and recent SOTA-style
architectures across tabular and multi-omics workflows.

Execution Notes
---------------

- Tabular benchmarks are typically run through the unified benchmark runner
  (registry-managed datasets and shared defaults)
- Multi-omics architectures are usually launched through dedicated scripts due
  to custom input formats

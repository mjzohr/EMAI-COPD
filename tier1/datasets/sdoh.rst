SDOH Dataset (COPDGene P2)
==========================

Dataset Details
---------------

Dataset Description
~~~~~~~~~~~~~~~~~~~
Social Determinants of Health (SDOH) features providing contextual information about participants.

Dataset Structure
-----------------
- Samples: 4,935
- Features: 34

Uses
-----

Direct Use
~~~~~~~~~~
- Contextual modeling
- Fairness and bias analysis

Out-of-Scope Use
~~~~~~~~~~~~~~~~
- Standalone predictive modeling (limited signal alone)

Bias, Risks, and Limitations
---------------------------
- High potential for societal bias
- Sensitive attributes may influence predictions

Recommendations
---------------
- Use carefully in fairness-aware modeling
- Analyze subgroup performance
